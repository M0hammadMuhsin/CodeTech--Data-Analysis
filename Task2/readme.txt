# Breast Cancer Prediction using Logistic Regression

This project uses the built-in breast cancer dataset from sklearn to train a logistic regression model that predicts whether a tumor is malignant or benign.

## Files

- `model.ipynb`: Jupyter notebook with EDA, model training, and evaluation.
- `requirements.txt`: List of Python dependencies.
- `dataset/`: Folder to store CSVs (if exported).
- `images/`: Folder to store generated plots.

## How to Run

```bash
pip install -r requirements.txt
jupyter notebook model.ipynb
