  PySpark Taxi Trip Data Analysis

This project demonstrates scalable big data analysis using PySpark on a synthetic taxi trip dataset.

  Tools Used
- PySpark
- Google Colab
- Pandas (optional)
- NYC-style trip data

  Insights
- Average fare per passenger
- Most common passenger count
- Longest & shortest trips

  Files
- `pyspark_analysis.ipynb` – Full analysis code
- `taxi_tripdata.csv` – Input dataset
- `report.md` – Final insights report
- `cleaned_taxi_data.zip` – Optional output
