# 📊 Superstore Sales Dashboard (Tableau)

## 📝 Project Overview
This project involves creating an interactive dashboard using **Tableau** to visualize and analyze the **Superstore Sales** dataset. The dashboard provides actionable insights into sales, profit trends, and customer behavior to assist in business decision-making.

---

## 📂 Dataset
- **Name**: Superstore_Sample.xlsx  
- **Source**: Sample dataset used widely for data visualization practice  
- **Fields Include**: Order ID, Order Date, Ship Date, Category, Sub-Category, Sales, Profit, Quantity, Region, State, Customer, and more.

---

## 🧰 Tools Used
- **Tableau Desktop** / **Tableau Public**
- Microsoft Excel (for initial dataset formatting)

---

## 📈 Dashboard Components

### 1. **Sales by Category**
- Bar chart showing sales volume for each product category.

### 2. **Profit Over Time**
- Line chart showing monthly profit trends across categories.

### 3. **Sales by State**
- Filled map visualizing total sales across U.S. states.

### 4. **Top Customers**
- Horizontal bar chart of top-performing customers by sales.

---

## 🧠 Actionable Insights
- 📉 Technology category drives revenue but has lower profit margins.
- 📈 Office Supplies sees consistent growth in specific states.
- ⚠️ Certain sub-categories (e.g., Phones) have high discounts and negative profit.
- 🥇 California leads in total sales; focus for premium offerings.
